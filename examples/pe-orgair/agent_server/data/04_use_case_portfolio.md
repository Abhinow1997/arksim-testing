# AI Use Case Portfolio Assessment

## Revenue-Generating AI Use Cases by Sector

### SaaS / Enterprise Software
- **Predictive churn modeling**: 15–25% reduction in customer churn
- **AI-powered search and recommendations**: 20–35% lift in feature adoption
- **Intelligent document processing**: 60–80% reduction in manual processing time
- **Conversational AI / chatbots**: 30–50% deflection of Tier-1 support tickets
- **Pricing optimization**: 5–15% gross margin improvement
- **Anomaly detection**: 40–60% faster incident resolution

### Financial Services
- **Credit risk scoring**: 10–20% improvement in default prediction accuracy
- **Fraud detection**: Real-time transaction scoring at <100ms latency
- **Portfolio optimization**: Sharpe ratio improvement through ML signals
- **KYC automation**: 70% reduction in onboarding time
- **Regulatory reporting**: Automated CCAR/DFAST stress testing

## Use Case Portfolio Scoring Framework

When evaluating a company's AI use case portfolio:

| Portfolio Tier | Characteristics |
|---------------|----------------|
| Nascent | 0–2 AI use cases; all in pilot phase; no production revenue impact |
| Developing | 3–5 use cases; 1–2 in production; limited measurable impact |
| Advancing | 6–10 use cases; 3–5 in production; documented ROI on 2+ cases |
| Leading | 10–20 use cases; majority in production; AI contributes >5% to revenue |
| Pioneering | 20+ use cases; AI is core to competitive moat; documented $100M+ impact |

## Key Questions for Due Diligence

1. What % of revenue is directly attributable to AI-powered products?
2. What is the cost savings from AI automation vs. manual processes?
3. How many models are in production vs. R&D?
4. What is the average time-to-production for new AI use cases?
5. Are use cases built on proprietary models or third-party APIs?
6. What is the model refresh cadence?
